Riya,a university student in Berlin wants to choose a suitable outfit for different occasions without spending too much time. 
she opens the “Wear It Right” website and enter a prompt such as “casual outfit for cold weather” or “formal outfit for a presentation.” 
The system allows the user to also input the clothes they already own, such as jackets, shoes, and shirts.
Based on both the prompt and the user’s wardrobe, the system provides outfit suggestions with matching combinations and styling tips.
However, the student often feels confused about how to combine their existing clothes properly and is unsure whether the outfit fits the weather or occasion. 
This leads to wasted time and lack of confidence. The system solves this problem by giving personalized outfit suggestions based on both the situation
and the user’s available clothes, helping her save time and make better decisions.

Objective: Choose outfit easily using available clothes
Actor: University student
Activities: Open website → select filters → add wardrobe → get suggestions
Problem: Confusion about combinations + limited wardrobe
Resolution: AI suggests outfits using user’s own clothes

One gap identified is that users may not want to manually enter all their clothes every time, which could make the system inconvenient. 
The system may need a feature to save or manage a user’s wardrobe.
